Common issues

  • Runs and territories require Location access while using the app.
  • Background route tracking works only during an active run.
  • Health sync requires Apple Health permission.
  • Cloud features require iCloud to be available on the device.

Account deletion

Groundwork supports in-app account deletion from the profile screen. Deleting your account permanently removes your profile, username, territories, and stored run history.
